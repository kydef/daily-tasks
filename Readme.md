# Daily tasks

This is a simple app for save routine tasks

## Setup

- Install git (details: https://git-scm.com/downloads/win)
- Run `git clone https://github.com/kydef/daily-tasks.git` inside your dev directory
- Open downloaded folder in your editor. 

## Development steps

- Implement basic actions: add and delete (details: https://uk.javascript.info/introduction-browser-events)
- Use local storage for save tasks (details: https://uk.javascript.info/localstorage)
- Add dark theme
